## Unit 1: LaTeX

Contains 3 folders, one for each assignment. The assignments are:
* Demonstrate the use of all the basic features of LaTeX in a single document by following the official tutorials provided on the Overleaf website.
* Using any IEEE Conference research paper as reference, prepare a document using LaTeX with identical content and formatting, following the template provided by IEEE.
* Prepare a presentation on any topic using the Beamer document class in LaTeX.

Each folder contains the relevant tex file, images included in the document and the PDF of the compiled code.

## Unit 2: Shell Scripting

Consists of 3 assignments-- A calculator, sum and average and to-do list

Run the bash files using:

`bash [file_name.sh]`

### Calculator:

Write a program to simulate a calculator, supporting operations like +, -, /, \*.

### Sum and Average

Write a shell program which reads a set of unspecified count of numbers and prints their sum and average.

### To-do List

Write a shell script to manage a todo list from command line. The script should be able to add, remove, list, sort, prepend, append, deduplicate todo items.

## Unit 3: Git
